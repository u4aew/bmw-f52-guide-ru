## Через бортовой компьютер (BC)

<ol class="steps">
  <li class="step">
    <span class="step-num">1</span>
    <span class="step-text">Включите зажигание (не заводя двигатель) или запустите автомобиль.</span>
  </li>
  <li class="step">
    <span class="step-num">2</span>
    <span class="step-text">На рычажке поворотника найдите кнопку бортового компьютера (BC) — нажмите её для входа в меню.</span>
  </li>
  <li class="step">
    <span class="step-num">3</span>
    <span class="step-text">Пролистайте до пункта <strong>«Техническое обслуживание»</strong> / <strong>Service</strong>.</span>
  </li>
  <li class="step">
    <span class="step-num">4</span>
    <span class="step-text">Удерживайте кнопку подтверждения <strong>3–5 секунд</strong> — появится запрос на сброс.</span>
  </li>
  <li class="step">
    <span class="step-num">5</span>
    <span class="step-text">Подтвердите сброс — счётчик обнулится до следующего интервала.</span>
  </li>
</ol>


:::tip[Когда делать сброс]
Сбрасывайте индикатор **сразу после замены масла** — до следующего запуска или в тот же день. Если сброс не сделан, машина продолжит отсчёт от старого интервала.
:::
